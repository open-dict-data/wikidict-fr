# wikidict-fr - Wikipedia Bilingual Reference Data (French)

This repository makes available parallelized interwiki link data from [Wikidata](https://www.wikidata.org/), which can be used, among other things, for the purpose of creating large offline French dictionaries.

## Data

The data directory contains subdirectories arranged in order of [ISO language code](http://en.wikipedia.org/wiki/ISO_639-1). The target language in each case is `fr` (French).

The basic filename pattern is `[ISO]-fr_wiki.txt`, with `[ISO]` being the source language ISO code. A list of all language pairs is [below](#available-language-pairs).
